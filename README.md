# МАГИСТРАЛ — сайт (Next.js + Tailwind)

Responsive сайт за комплекс с бензиностанция, автогаз, пълнене на газови бутилки,
строителна борса, сервиз за гуми, автомивка и магазин за авто аксесоари.

## Стартиране

Трябва да имаш инсталиран **Node.js 18+**.

```bash
npm install      # инсталира зависимостите (иска интернет)
npm run dev      # стартира на http://localhost:3000
```

За реален сайт:

```bash
npm run build
npm start
```

## Как да сложиш своите 15 снимки

Всички снимки стоят в папка **`public/images/`**. Вече има 15 брандирани
заместващи картинки. Просто ги замени със своите, като запазиш **същите имена**:

| Файл | Къде се показва |
|------|-----------------|
| `00-hero.jpg` | Голямата снимка най-горе |
| `01-borsa.jpg` | Строителна борса |
| `02-benzinostanciya.jpg` | Бензиностанция |
| `03-butilki-gaz.jpg` | Пълнене на газови бутилки |
| `04-serviz-gumi.jpg` | Сервиз за гуми |
| `05-avtomivka.jpg` | Автомивка |
| `06-magazin.jpg` | Магазин за аксесоари |
| `g-07.jpg` … `g-14.jpg` | Галерия (8 снимки) |

Съвет: снимките да са водоравни (landscape). Хиро снимката – по-широка (напр. 1920×1080).

## Какво да смениш с твоите данни

Отвори **`lib/data.ts`** — всичко важно е там:

- **Име на фирмата** — полето `name` (сега пише „МАГИСТРАЛ").
- **Телефон** — `phone` и `phoneHref`.
- **Адрес** — `address`.
- **Работно време** — `hours`.
- **Цени на горивата** — `fuels` (сега са примерни `0.00`).
- **Текстове на услугите** — `services`.
- **Google карта** — сложи линк за вграждане в `mapEmbed`.

### Как се взима линк за Google картата
Google Maps → търсиш адреса → **Сподели** → **Вгради карта** → копираш
само адреса от `src="..."` и го слагаш в `mapEmbed`.

## Технологии

- Tailwind CSS
- Шрифтове: Oswald, Inter, IBM Plex Mono (с кирилица)
